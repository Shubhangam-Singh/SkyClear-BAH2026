The Problem

Clouds don't just block a view — they erase a moment

Persistent cloud cover over tropical and mountainous regions of India — the North Eastern Region especially — regularly blinds optical satellites like LISS-IV, and it tends to happen exactly when a clear image matters most: disaster response, monsoon-season monitoring, land-use mapping.

Traditional cloud masking doesn't solve this. It simply discards the obscured pixels, leaving an incomplete scene — even though the satellite already made the pass.

How It Works

From obscured to analysis-ready, in one pass

  1. 01

    Cloudy LISS-IV Input

    A satellite scene with regions obscured by real atmospheric cloud cover.

  2. 02

    Cloud Detection & Masking

    Cloud-affected regions within the scene are identified.

  3. 03

    GAN Reconstruction

    A U-Net generator and PatchGAN discriminator reconstruct the ground hidden beneath.

  4. 04

    Analysis-Ready Output

    A complete scene — no discarded data, no extra satellite pass required.

The key insight: naturally-paired cloudy/clean images of the same location barely exist. So SkyClear synthesizes realistic fractal clouds — soft edges, variable opacity, simulated shadows — over real cloud-free LISS-IV scenes, producing unlimited (cloudy, clean) training pairs without waiting on rare paired acquisitions.
